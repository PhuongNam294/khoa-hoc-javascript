- [ ] Hàm `alert`: được sử dụng để hiển thị thông báo dạng popup (hộp thông báo) trên trình duyệt người dùng.

  Cú pháp:  `alert("Nội dung thông báo");`
- [ ] Hàm `console.log` dùng để xuất thông tin ra màn hình điều khiển (console) của trình duyệt, ngoài ra dùng đẻ debug giúp phát hiện lỗi trong code.

  Cú pháp: `console.log(nội dung);`

  **Trong đó:**
  * `console`: Là đối tượng hệ thống.
  * `.log`: Là hành động (hàm) để in ra.
  * `( )`: Chứa thứ bạn muốn in (biến, số, hoặc chữ).
  * `" "`: Nếu là chữ (chuỗi), bạn phải đặt trong dấu ngoặc kép.

Lưu ý: Nếu code trên trình duyệt Google Chrome nếu bật Developer Tools (Ctrl + Shift + J) nếu muốn code nhiều dòng chạy cùng lúc thì nhấn Shift + Enter để xuống dòng code tiếp
